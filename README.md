# OpenCV 基础图像处理示例 (Image Resizing) 🖼️

本项目是一个基于 C++ 和 OpenCV 库的入门级示例程序，展示了如何加载图像并进行几何尺寸变换。

## 🚀 功能特性

* **控制台交互**：启动时输出欢迎信息 `Hello Open CV!`。
* **图像读取**：从本地路径加载目标图片（默认为 `123.jpg`）。
* **比例缩放**：通过线性插值算法（INTER_LINEAR），将原始图像的长宽均等比例缩小至 **25% (1/4)**。
* **可视化展示**：创建一个交互式窗口显示处理后的缩小图像。

## 🛠️ 核心代码逻辑

程序使用了 OpenCV 的 `cv::resize` 函数，其核心调用如下：
```cpp
// 参数 0.25 表示在 X 轴和 Y 轴方向缩放比例为 1/4
resize(temImage, dstImage1, Size(0, 0), 0.25, 0.25, INTER_LINEAR);
